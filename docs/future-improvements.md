# Future Improvements

## Overview

This Power BI dashboard could be extended in several ways to support deeper analysis and improve usability.

## Refresh with Later 2026 Data

One future improvement would be to refresh the analysis with later 2026 data, especially July 2026 or later.

This would make it possible to check whether outbreak patterns continued to decrease and whether later months showed any change after major public events.

## Dynamic Subtitle

Another improvement would be to add a dynamic subtitle that updates based on the selected Month Year filter.

For example, the dashboard subtitle could automatically show:

```text id="zhsm3l"
View: Jan 2026 - Jun 2026
```

This would make it clearer which time period is reflected in the KPI cards and visuals.

## Additional Dashboard Pages

Additional dashboard pages could be created for deeper analysis, such as:

* a historical trend page comparing multiple years
* a COVID-19 specific trend page
* a causative agent analysis page
* an institution-level monitoring page
* a duration analysis page focused on long-running outbreaks

## Geographic Analysis

The dashboard could also be expanded with geographic analysis if address-level data is cleaned and prepared for mapping.

This could help identify whether outbreaks are concentrated in specific areas of Toronto.

## Report Sharing

Another future improvement would be to publish the report online if Power BI licensing and sharing options allow it.

This would make the dashboard easier to explore interactively instead of relying only on screenshots and the `.pbix` file.

## Summary

These improvements would make the project stronger as both an analytical dashboard and a portfolio project.
